바로고 요금제 변경 안내

바로고 요금제 주의사항

  1. 1. 바로고 요금제에서 일반 요금제로 변경 시 주유권 지급은 중단 됩니다.
    또한 다시 바로고 요금제로 변경이 불가하오니 이점 유의하시기 바랍니다.
    (단, 바로고 요금제간 변경은 가능합니다.)
  2. 2. 정지 시 주유권 지급은 중단되며 재사용 익월 주유권이 지급됩니다.
  3. 3. 정지 기간은 쿠폰지급 가능 기간에 포함 됩니다.
  4. 4. 해지 시 주유권 지급은 중단됩니다.

쿠폰사용 유의사항

쿠폰은 지급일로 부터 30일간 사용이 가능합니다.
30일 이후 쿠폰은 자동 소멸 되오니 이점 유의하시기 바랍니다.
