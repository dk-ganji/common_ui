카앤피플 요금제 변경 안내

카앤피플 요금제에서 타 요금제로 변경 시 카앤피플 포인트 쿠폰 지급은 즉시 중단됩니다.
또한 다시 카앤피플 요금제로 재변경이 불가하오니, 이점 유의하시기 바랍니다.

꼭! 알아두세요.

제공 세차 포인트 유의사항
  • 카앤피플 요금제 전월 정상 이용 시에만 개통한 번호로만 쿠폰수신 쿠폰 발송됩니다.
    (명의변경 / 요금제 변경 / 정지 / 해지 등 비정상 이용 시 대상 제외)
  • 포인트 쿠폰은 개통 후 24개월까지 총 12회(2개월 마다) 제공됩니다.
  • 포인트 쿠폰 최초 발송은 개통한 익월(D+1월) 15일에 문자로 일괄 발송 됩니다.
    (예시. 21년 4월 개통 완료 → 21년 5월 15일 최초 발송, 이후 격월 15일 발송)
  • 발행된 포인트 쿠폰은 카앤피플 전용 App을 통하여 사용 가능합니다. (회원가입 필요)
  • 발행된 포인트 쿠폰의 사용 유효기간은 1년입니다.
    (예시. 21년 5월 15일 쿠폰 수령 → 22년 5월 14일 포인트 자동소멸)
  • 발행 된 포인트 쿠폰의 분실 및 삭제에 따른 재발송은 불가합니다.
  • 요금제 개통 시, 카앤피플 ‘서비스 가능 지역’이 → ‘불가능 지역’으로 변경 시에는 ‘서비스 가능
    지역’에서만 사용 가능합니다.
    (출장 세차 서비스 가능지역 변경에 따른 불이익은 U+유모바일과 카앤피플에서는 책임지지 않습니다.)
