휴대폰 정보 등록

유심요금제에 가입하셨다면 휴대폰 정보 등록으로 더 완전한 서비스 이용하세요.

휴대폰 정보 등록이란?

U+알뜰모바일에서 구매한 휴대폰(새폰/중고폰) 외 자급제, 중고 거래로 구매한 휴대폰의 소유권을 등록하는거에요.

휴대폰 정보 등록하면 이런게 좋아요.

  • 자급제, 중고거래로 구매한
    휴대폰의 분실/파손보험 가입 가능

  • 휴대폰 분실 신고 가능

  • 번호 변경 가능

  • 유심만 개통 후 일부 서비스 이용이
    안되는 문제 해결

등록 방법

  1. 1. 휴대폰 정보 변경 화면에 진입합니다.
  2. 2. 휴대폰 등록을 위해서는 본인 확인이 필요합니다. 원하는 인증 수단을 선택해 진행해 주세요.
  3. 3. 본인인증 후 본인 정보와 휴대폰 정보를 입력합니다.
  4. 4. 휴대폰의 모델명, 일련번호, MAC 주소, IMEI는 우측의 확인 방법을 눌러서 확인해 보세요.
    • 모델명/일련번호 확인 방법 보기
    • MAC 주소/IMEI 값 확인 방법 보기
  5. 5. 휴대폰 등록이 완료되었습니다.
