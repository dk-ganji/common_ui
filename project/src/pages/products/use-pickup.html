기다리지 않고 편의점에서 유심을 바로 교환하세요.

바로픽업이란?

바로픽업 방법

  1. 1. 유심카드 구매하기 화면에 진입합니다.
  2. 2. 배송 방법 중 원하는 편의점의
    바로 픽업을 선택합니다.
  3. 3. 유심 교환권을 받을 수령인 정보를 입력 후 인증합니다.
  4. 4. 결제하기 전 주소나 매장명을 검색해
    가까운 픽업 장소를 확인해 보세요.
  5. 5. 원하는 결제수단을 선택 후 결제를 완료합니다.
  6. 6. 결제가 완료되면 입력한 휴대폰번호로 모바일 쿠폰이 발송됩니다.
  7. 7. 매장에 방문하여 교환권으로 유심을
    픽업해보세요.
