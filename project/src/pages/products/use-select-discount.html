할인 반환금 안내

1. 할인 반환금 부과 기준

아래와 같은 경우 반환금이 발생됩니다.

  • 약정기간 중에 개통을 해지하거나 기타 약정 할인 혜택 이용을 취소하는 경우
  • 할인 적용 안 되는 요금제로 변경하는 경우
  • 유심(개인 정보 저장)카드 기변을 제외하고, 선택 약정 할인(통신 요금 25% 할인)이 적용되지 않는 휴대폰으로 변경하는 경우
  • 명의변경 시 약정을 이어 받지 않는 경우

아래와 같은 경우 할인반환금을 청구하지 않습니다.

  • 고객의 주 생활지의 통화 품질이 불량하여 개통일로 부터 14일 내에 해지한 경우
  • 단, 구입 및 제공받은 기기 구성품 모두를 손상 없이 반납한 경우에 한함
  • 고객의 사망으로 인한 해지

유의사항

  • 월 중 가입, 요금제를 변경하는 경우는 사용 일수만큼 계산합니다.
  • 약정 기간 설정 후 약정 기간 변경이 불가능합니다.
  • 선택약정 할인(통신 요금 25% 할인) 기준할인율은 가입 당시 정책에 따르며, 약정 만료일까지 유지됩니다.
  • 가입 당시에는 없는 신설 요금제로 요금제 변경하는 경우 가입 당시의 ‘선택약정 할인(통신 요금 25% 할인)’ 정책을 적용합니다. (가입 당시의 기준할인율을 적용)
  • 명의변경 시 약정 승계하는 경우 가입 당시의 ‘선택약정 할인(통신 요금 25% 할인)’ 정책을 적용합니다. (미승계시 기존 명의자에게 선택약정 할인(통신 요금 25% 할인)반환금이 청구됨)
  • 정지 기간은 약정 이용 기간에서 제외되며 해당 기간만큼 약정 만료일(기간) 연장, 일시/이용정지 요금 할인 대상에서 제외 됩니다.
  • 요금할인은 요금제 월정액에 대해서만 할인이 가능합니다.
  • ‘선택약정 할인(통신 요금 25% 할인)’의 약정기간(24개월)을 사용 후 다시 재 약정하여 선택약정 할인(통신 요금 25% 할인) 가입 시 재 약정 당시 기준할인율을 적용합니다.
  • 선택약정 할인(통신 요금 25% 할인) 가입 가능 조회 및 가입 신청은 로그인 후 이용하실 수 있으며 가입신청은 직영대리점, 고객센터, 홈페이지에서 가능합니다.
  • 기존 장기고객할인 25%은 선택약정 할인(통신 요금 25% 할인)으로 명칭이 변경됩니다.
