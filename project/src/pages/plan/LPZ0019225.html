기본 혜택

U+와이파이

매너콜

핫스팟

  • 기본제공 데이터가 줄어들며, 기본 제공 데이터를 초과하여 사용하는 경우 요금이 발생

이용조건

  • LTE eSIM만 개통이 가능하고, 고객이 소유한 휴대폰을 이용해 개통이 가능합니다.

데이터 이용 안내

  • 3GB 이하 사용하는 경우 : 22.53원/MB로 요금 발생(19,800원 이상은 부과되지 않음)
  • 3GB 초과 사용하는 경우 : 6.76원/MB로 요금 발생
    정보이용료 별도 발생됩니다.

통화 이용 안내

  • 유무선 기본제공이란?
    무선/유선/인터넷전화 : 정상 이용 시 사용량에 관계없이 무료로 제공
  • 무선(010,011,016,017,018,019), 유선(지역번호 02,03X,04X,05X,06X), 인터넷전화(070)
    그 외 통화 : 합산하여 아래의 제공 통화량 내에서 이용가능
  • 그 외 통화 : 무선/유선/인터넷전화가 아닌 모든 통화(전국대표번호(15XX,16XX), 평생개인번호(050X), 주파수공용통신(013) 등)

문자 이용 안내

  • 사업자 구분 없이 무제한 제공됩니다.
    단, 스팸, 상업적 용도 사용, 과도한 사용을 방지하기 위해서 아래의 경우 회사는 정상요금 부과,이용정지 등을 적용할 수 있습니다.
  • 하루 150건 초과하는 문자메시지를 월 10회 초과하여 보내는 경우
  • 하루 500건을 초과하는 문자메시지를 보내는 경우
  • 한 달에 2,000개를 초과하는 번호로 문자메시지를 보내는 경우
  • 휴대폰 이외의 시스템을 통해 문자메시지를 발송하는 경우
  • 제 3자에게 휴대전화를 임대하는 등 이용약관을 위반하는 경우

유의 사항

월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 사용일 수만큼 계산하여 적용
  • 프로모션 요금할인 내역은 개통 익월 15일 이후부터 홈페이지 및 고객센터 앱을 통해 확인 가능합니다.

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함 (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
