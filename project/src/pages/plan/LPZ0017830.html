기본 혜택

U+와이파이

매너콜

핫스팟

  • 데이터 월 10GB 별도 제공
  • 모두 사용하면 자동 차단

이용 조건

  • 5G 휴대폰 신규, 기기변경 시 이용 가능합니다.
  • 일부 지역에서는 5G서비스이용이 불가능 할 수도 있습니다. 이때에는 LTE서비스로 자동 전환됩니다.
  • 5G 요금제와 LTE 요금제간 변경은 불가합니다. (상세한 내용은 고객센터로 문의하세요.)
  • 요금제 변경 : 5G 휴대폰 요금제로만 변경 가능합니다.

데이터 이용 안내

  • 3GB 이하 사용하는 경우: 22.53원/MB로 요금 발생(19,800원 이상은 부과되지 않음)
  • 3GB 초과 사용하는 경우: 6.76원/MB로 요금 발생
  • 정보이용료 별도 발생됩니다.

통화 이용 안내

  • 음성통화 : 1초 → 1초 / 소진 후 1초당 1.98원
  • 영상통화 : 1초 → 1.66초 / 소진 후 1초당 3.3원
  • 무료 통화 및 무료 문자메시지는 국내 통화에 한하여 제공되며, 미소진 잔여량은 이월되지 않습니다.
    또한 월 단위로 제공되므로, 월 중 요금제 가입/변경/해지시 요금제 사용일만큼 계산되어 제공됩니다.
  • 음성 및 동영상 통화는 통화 시작 시간 기준으로 요금이 발생됩니다.(월말 시작하여 익월 종료된 통화는 통화 시작월에 요금 발생(무료통화 소진)
  • 음성 및 동영상 통화는 통화 시작 시간 기준으로 요금이 발생됩니다. (월말 시작하여 익월 종료된 통화는 통화 시작월에 요금 발생됩니다. (무료통화 소진)
  • 기본제공 데이터통화량은 순수 데이터통화용으로만 사용이 가능하며 본 요금제 데이터서비스를 이용한 음성통화/동영상통화는 사용할 수 없습니다.

문자 이용 안내

  • 단문메시지(SMS) : 22원 / 한글 70자(140Byte)이하
  • 장문메시지(LMS) : 22원 / 한글 41~120자(81~240Byte)
  • 멀티메시지(MMS) :
    44원 / 전송용량 141~2,000Byte
    220원 / 그림/배경음악 첨부
    440원 / 동영상 첨부

유의 사항

월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 일할계산하여 적용
  • 프로모션 요금할인 내역은 개통 익월 15일 이후부터 홈페이지 및 고객센터 앱을 통해 확인 가능합니다.

요금제의 변경

  • 요금제의 변경 신청을 하는 경우에는 변경 신청일 당일부터 변경된 요금을 적용함 (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
  • 요금 할인이 적용된 요금제를 이용하던 중, 타 요금할인 요금제로 변경 시 매월 1일에 변경/예약해야 정상 요금할인 적용이 가능함 (월 중 요금제 변경 시 현재 사용 중인 요금제의 요금할인 적용 불가능)
  • 요금제 내 별도의 명시가 없는 경우 기본 제공 음성, 문자메시지, 데이터 통화 등은 국내에 한해 제공됩니다.
  • 현재 5G 서비스는 일부 지역에서는 제공되지 않을 수 있으며, 계속 확대해 나갈 예정입니다. (5G 서비스 미제공 지역은 LTE로 제공)
  • 5G 요금제 및 5G 부가상품 가입 시 제공되는 데이터 용량은 4G(LTE) 구분 없이 각 N/W 제공 가능 범위에 따라 제공됩니다.
