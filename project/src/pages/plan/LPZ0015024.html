U+와이파이

매너콜

핫스팟

데이터 이용 안내

  • 월 기본 제공량 소진 시, 속도제한 적용되며 요금 부과되지 않습니다.
  • 월 기본 제공량 15GB 소진 후 최대 3Mbps 속도로 계속 이용 할 수 있습니다.
  • 테더링 이용(테더링 : 내 휴대폰과 연결하여 다른 기기에서 와이파이 사용)
  • 테더링을 이용하면 데이터 사용은 월 제공데이터(15GB)내에서 이용이 가능합니다.

통화 이용 안내

  • 음성통화 : 1초 → 1초 / 소진 후 1초당 1.98원
  • 영상통화 : 1초 → 1.66초 / 소진 후 1초당 3.3원
  • 월 제공량 미소진 잔여량은 이월되지 않습니다.
    또한 월 단위로 제공되므로,월 중 요금제 가입/변경/해지 시 요금제 사용일만큼 계산되어 제공됩니다.
  • 음성 및 영상 통화는 통화 시작 시간 기준으로 적용됩니다.
    예시 : 월말 시작하여 익월 종료된 통화는 통화 시작월 기준으로 적용됩니다. (잔여 제공량이 있을 경우 차감되며, 이후 사용량은 과금)

문자 이용 안내

  • 단문메시지(SMS) : 22원 / 한글 70자(140Byte)이하
  • 장문메시지(LMS) : 22원 / 한글 41~120자(81~240Byte)
  • 멀티메시지(MMS) :
    44원 / 전송용량 141~2,000Byte
    220원 / 그림/배경음악 첨부
    440원 / 동영상 첨부

유의 사항

  • 월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.
  • 통신사를 바꾸거나 해지한 경우, 기존에 등록한 LG U+ 망내 1회선 음성무제한 번호는 자동 삭제되며, 재등록이 이루어져야 할인적용 가능합니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 일할 계산하여 적용

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함 (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
  • 요금제별 단말지원금 등 혜택이 상이할 수 있기때문에, 최초 신청한 요금제는 약정기간내 타요금제로 변경한 경우 수수료가 부과될 수 있음 ( 단, 최초 받은 혜택 범위 내에서 수수료부과)
  • 요금제 내 별도의 명시가 없는 경우 기본제공 음성, 문자, 데이터통화 등은 국내에 한해 제공됩니다.
  • 현재 4G(LTE)서비스는 일부 지역에서는 제공되지 않을 수 있으며, 계속 확대해 나갈 예정입니다.
  • LTE 요금제 및 LTE 부가상품 가입 시 제공되는 데이터 용량은 3G와 4G(LTE)구분 없이 각 N/W 제공 가능 범위에 따라 제공됩니다.
