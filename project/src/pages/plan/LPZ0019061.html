기본 혜택

U+와이파이

매너콜

핫스팟

  • 데이터 월 10GB 별도 제공
    모두 사용하면 자동 차단

이용조건

  • USIM만 개통이 가능하고, 고객이 소유한 5G 휴대폰으로 이용이 가능합니다.
  • 일부LTE휴대폰으로도 사용이 가능하나, 5G망 사용은 불가능합니다. (LTE속도로만 사용가능)
  • 일부 지역에서는 5G서비스이용이 불가능 할 수도 있습니다. 이때에는 LTE서비스로 자동 전환됩니다.
  • 5G 요금제와 LTE 요금제간 변경은 불가합니다. (상세한 내용은 고객센터로 문의하세요.)
  • 2020년 11월 15일부로 LTE 자급제폰에서도 사용가능

데이터 이용 안내

  • 기본제공 데이터에서만 mVoIP 이용 가능하며 테더링 사용가능(기본제공 데이터 모두 소진 시 차단)
  • 기본 제공량을 모두 사용하면 문자메시지로 알려드리며 이후 통화료는 부가세가 포함된 1MB당 22.53원이 발생합니다.
  • 3GB 이하 사용하는 경우 : 22.53원/MB로 과금 (19,800원 이상은 부과되지 않음)
  • 3GB 초과 사용하는 경우 : 6.76원/MB로 과금

통화 이용 안내

  • 음성통화 : 1초 → 1초 / 소진 후 1초당 1.98원
  • 영상통화 : 1초 → 1.66초 / 소진 후 1초당 3.3원

문자 이용 안내

  • 단문메시지(SMS) : 22원 / 한글 70자(140Byte)이하
  • 장문메시지(LMS) : 22원 / 한글 41~120자(81~240Byte)
  • 멀티메시지(MMS) :
    44원 / 전송용량 141~2,000Byte
    220원 / 그림/배경음악 첨부
    440원 / 동영상 첨부

유의 사항

  • 네이버페이 포인트는 신규가입 및 번호이동 고객에 한해 적립됩니다.
  • 네이버페이 포인트는 기본요금(부가세 제외)의 10%가 가입하신 네이버ID로 매월 최대 24개월까지 적립됩니다.
  • 10% 적립 요금제 외 요금제로 변경, 명의변경, 해지 시 포인트 적립은 즉시 중단되며, 다시 10% 적립 요금제로 변경이 불가능합니다.
    (단, 요금제 사용 일자까지 일할 계산하여 적립됩니다.)
  • 10% 적립 요금제에 중도 가입 시 포인트는 사용한 일수에 따라 일할 계산되어 적립됩니다.
  • 네이버페이 포인트는 요금제 사용 익월 말일에 일괄 적립되며, 적립시점에 네이버페이를 탈퇴하거나 보유 포인트가 200만원을 초과한 경우 지급이 불가합니다.
  • 네이버페이 포인트 지급 내역은 네이버>네이버페이>네이버페이 포인트에서 확인 가능합니다.
  • 월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 일할 계산하여 적용
  • 요금할인 내역은 개통 익월 15일 이후부터 홈페이지 및 고객센터 앱을 통해 확인 가능합니다.

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함
    (단, 요금제 변경일로부터 1개월 이 내에는 재변경 불가)
  • 요금제별 단말기 요금 등 혜택이 상이 할 수 있기 때문에, 최근 신청한 요금제는 약정기간 내 타 요금제로 변경할 경우 수수료가 부과 될 수 있음
    (단, 최근 받은 혜택 범위 내에서 수수료부과)
  • 5G 요금제와 LTE 요금제간 변경은 불가합니다. (상세한 내용은 고객센터로 문의하세요.)
  • 2020년 11월 15일부로 LTE 자급제폰에서도 사용가능
  • 5G 요금제는 일부LTE휴대폰으로도 사용이 가능하나, 5G망 사용은 불가능합니다. (LTE속도로만 사용가능)
  • 일부 지역에서는 5G서비스이용이 불가능 할 수도 있습니다. 이때에는 LTE서비스로 자동 전환됩니다.
  • 요금제 내 별도의 명기가 없는 경우 기본제공 음성, 문자, 데이터통화 등은 국내에 한해 제공됩니다.
  • 10% 적립 요금제는 (주)미디어로그에서 네이버파이낸셜㈜에 위탁하여 판매하며, 10% 적립 요금제의 서비스 운영 관리 및 계약은 (주)미디어로그(U+유모바일)와 이루어 집니다.
