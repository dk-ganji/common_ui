데이터 이용 안내

  • 기본 제공량을 모두 사용하면 문자메시지로 알려드리며 이후 통화료는 다음과 같이 부과 됩니다.
  • 기본 제공량을 모두 사용한 후 초과 사용 6G까지 MB 당 14.08원 부과지만 27,500원 이상은 부과되지 않습니다.
  • 초과 사용 6G 이후에는 다시 MB 당 14.08원으로 추가 요금이 발생합니다.

통화 이용 안내

  • 음성통화 : 1초 → 1초 / 소진 후 1초당 1.98원
  • 영상통화 : 1초 → 1.66초 / 소진 후 1초당 3.3원
  • 월 제공량 미소진 잔여량은 이월되지 않습니다.
  • 또한 월 단위로 제공되므로,월 중 요금제 가입/변경/해지 시 요금제 사용일만큼 계산되어 제공됩니다.
  • 음성 및 영상 통화는 통화 시작 시간 기준으로 적용됩니다.
  • 예시 : 월말 시작하여 익월 종료된 통화는 통화 시작월 기준으로 적용됩니다.
    (잔여 제공량이 있을 경우 차감되며, 이후 사용량은 과금)

문자 이용 안내

  • 단문메시지(SMS) : 22원 / 한글 70자(140Byte)이하
  • 장문메시지(LMS) : 22원 / 한글 41~120자(81~240Byte)
  • 멀티메시지(MMS) :
    44원 / 전송용량 141~2,000Byte
    220원 / 그림/배경음악 첨부
    440원 / 동영상 첨부

유의 사항

월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 일할 계산하여 적용

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함
    (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
  • 요금제별 단말기 요금 등 혜택이 상이 할 수 있기 때문에, 최근 신청한 요금제는 약정기간 내 타 요금제로 변경할 경우 수수료가 부과 될 수 있음
    (※ 단, 최근 받은 혜택 범위 내에서 수수료부과)
  • 요금제 내 별도의 명기가 없는 경우 기본제공 음성, 문자, 데이터통화 등은 국내에 한해 제공됩니다.
