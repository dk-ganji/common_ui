기본 혜택

U+와이파이

매너콜

핫스팟

  • 데이터 월 11GB 소진 시 일2GB 순으로 데이터 차감 모두 사용하면 자동 차단

이용조건

  • 파손형 보험 (폰 파손보험 25 또는 I폰 파손보험 30) 24개월 지원(요금 할인) & 파손비의 자기부담금 U+유모바일 포인트로 리워드
  • ※ 자기부담금 : 휴대폰 수리 시 손해액의 30% (최소 3만원)
  • ※ 자급제 아이폰 또는 갤럭시 단말로 자급제폰케어 요금제 가입 후 별도로 보험 가입을 해주셔야 보험 혜택이 지급됩니다.

데이터 이용 안내

  • 3Mbps의 속도는 동영상(720P), 인터넷검색, 인*타그램(SNS)등을 무리없이 이용 가능한 속도 (단, 네트워크 상황에 따라 달라질 수 있음)
  • * 데이터테더링(핫스팟)은 기본제공량 소진 시 이용 불가능합니다.

통화 이용 안내

  • 집전화 이동전화 무제한
  • 부가통화 300분
  • 부가통화 1초 이용 시 부가통화 제공량에서 1초 줄어듦
  • 영상통화 1초 이용 시 부가통화 제공량에서 1.66초 줄어듦
  • 아래의 경우에 해당 시, 본 요금제의 이용을 제한할 수 있음
  • 일 음성 통화량이 600분을 초과하는 횟수가 월 중 3회를 넘는 경우
  • 당월 총 음성 통화량이 6,000분을 초과하는 경우
  • 텔레마케팅 / 폰팅 등과 같이 음성통화를 광고/상업적 목적으로 이용하는 경우

문자 이용 안내

  • 아래의 경우에 해당 시, 본 요금제의 이용을 제한할 수 있음
  • 하루 150건 초과하는 문자메시지를 월 10회 초과하여 보내는 경우
  • 하루 500건을 초과하는 문자메시지를 보내는 경우
  • 한 달에 2,000개를 초과하는 번호로 문자메시지를 보내는 경우
  • 휴대폰 이외의 시스템을 통해 문자메시지를 발송하는 경우
  • 제 3자에게 휴대전화를 임대하는 등 이용약관을 위반하는 경우

유의 사항

  • 월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는 사용일 수만큼 계산하여 적용
  • 프로모션 요금할인 내역은 개통 익월 15일 이후부터 홈페이지 및 고객센터 앱을 통해 확인 가능합니다.

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함 (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
  • 요금 할인이 적용된 요금제를 이용하던 중, 타 요금할인 요금제로 변경 시 매월 1일에 변경/예약해야 정상 요금할인 적용이 가능함
    (월 중 요금제 변경 시 현재 사용 중인 요금제의 요금할인 적용 불가능)
  • 보험 제공 혜택 기간 중 요금제 변경 시 보험 제공 혜택 지급은 중단되며, 보험료는 요금제 변경일로부터 일할 계산되어 청구됩니다.
  • 5G 요금제와 LTE 요금제간 변경은 불가합니다. (상세한 내용은 고객센터로 문의하세요.)
    ※ 2020년 11월 15일부로 LTE 자급제폰에서도 사용가능
  • 5G 요금제는 일부LTE휴대폰으로도 사용이 가능하나, 5G망 사용은 불가능합니다.
    (LTE속도로만 사용가능)
  • 일부 지역에서는 5G서비스이용이 불가능 할 수도 있습니다. 이때에는 LTE서비스로 자동 전환됩니다.
  • 요금제 내 별도의 명시가 없는 경우 기본제공 음성, 문자메시지, 데이터통화 등은 국내에 한해 제공됩니다.
  • 보험 보상 관련 문의는 보상센터 (1644-9471, 평일 09~18시)를 통해 가능합니다.
  • 자급제 아이폰이 아닌 경우 보험 제공 혜택을 받으실 수 없습니다.
  • 보험 제공 혜택은 보험 가입자를 대상으로 지급됩니다. U+유모바일 가입 후 고객센터/홈페이지를 통해 보험 가입 신청을 하셔야 합니다.
