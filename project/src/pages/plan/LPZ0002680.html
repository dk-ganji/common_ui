기본 혜택

U+와이파이

매너콜

핫스팟

  • 데이터 월 11GB 소진 시 일2GB 순으로 데이터 차감
    모두 사용하면 자동 차단

이용조건

  • 사용하지 못한 데이터는 포인트로 돌려드립니다.
    기준 : 남은 데이터 500MB당 100포인트 제공 (500MB이하는 미제공)
  • 포인트 지급일 : 사용월 + 2개월 후 지급
  • 포인트 지급 기준 데이터는 기본제공 데이터 입니다.
  • 프로모션으로 제공되는 추가 데이터가 남아도 포인트는 지급되지 않습니다.

데이터 이용 안내

  • 3Mbps의 속도는 동영상(720P), 인터넷검색, 인*타그램(SNS)등을 무리없이 이용 가능한 속도 (단, 네트워크 상황에 따라 달라질 수 있음)
  • 데이터테더링(핫스팟)은 기본제공량 소진 시 이용 불가능합니다.

통화 이용 안내

  • 집전화 이동전화 무제한
  • 부가통화 300분
  • 부가통화 1초 이용 시 부가통화 제공량에서 1초 줄어듦
  • 영상통화 1초 이용 시 부가통화 제공량에서 1.66초 줄어듦
  • 아래의 경우에 해당 시, 본 요금제의 이용을 제한할 수 있음
  • 일 음성 통화량이 600분을 초과하는 횟수가 월 중 3회를 넘는 경우
  • 당월 총 음성 통화량이 6,000분을 초과하는 경우
  • 텔레마케팅 / 폰팅 등과 같이 음성통화를 광고/상업적 목적으로 이용하는 경우

문자 이용 안내

  • 아래의 경우에 해당 시, 본 요금제의 이용을 제한할 수 있음
  • 하루 150건 초과하는 문자메시지를 월 10회 초과하여 보내는 경우
  • 하루 500건을 초과하는 문자메시지를 보내는 경우
  • 한 달에 2,000개를 초과하는 번호로 문자메시지를 보내는 경우
  • 휴대폰 이외의 시스템을 통해 문자메시지를 발송하는 경우
  • 제 3자에게 휴대전화를 임대하는 등 이용약관을 위반하는 경우

유의 사항

  • 본 요금제는 만 19세 이상부터 만 29세 이하만 가입이 가능합니다.
    - 1992년 주민번호 기준 생일이 안 지난 경우 가입 가능
    - 2003년 주민번호 기준 생일이 지난 경우 가입 가능
    - 이십세 요금제를 사용하다 30세가 경과한 경우에도 계속 사용 할 수 있습니다.
    단, 타 요금제로 변경하셨다가 다시 이십세 요금제는 이용이 불가능 합니다.
  • 월정액(기본료) 및 국내통화료에는 별도의 규정이 없으면 부가세가 별도로 청구됩니다.
  • 데이터 사용이 과다 할 경우 고객의 전화번호로 문자 또는 연락을 취할 수 있고 타 요금제로 변경 등을 안내 할 수 있습니다.
  • 기준 (제공량의 두배이상을 사용 하는 경우)
    * 이십세 5GB : 매월 데이터 사용량이 10GB 이상일 경우
    * 이십세 10GB : 매월 데이터 사용량이 20GB 이상일 경우

요금 등의 산정 기준일

  • 전월 1일부터 전월 말일까지
  • 요금월의 중도에 개통, 해지, 일시정지, 이용휴지 및 이용정지 되거나 요금제를 변경하는 경우에는, 사용일 수만큼 계산하여 적용
  • 요금할인 내역은 개통 익월 15일 이후부터 홈페이지 및 고객센터 앱을 통해 확인 가능합니다.

요금제의 변경

  • 요금제의 변경신청을 하는 경우에는 변경신청일 당일부터 변경된 요금을 적용함
    (단, 요금제 변경일로부터 1개월 이내에는 재변경 불가)
  • 요금 할인이 적용된 요금제를 이용하던 중, 타 요금할인 요금제로 변경 시 매월 1일에 변경/예약해야 정상 요금할인 적용이 가능함
  • 일반 요금제에서 이십세 요금제로 변경 하실 경우 매월 1일에만 변경이 가능 합니다. (고객센터로 요금제 변경 예약을 해주세요)
  • 남은 데이터 보상 포인트 제공 기준
    포인트는 500MB당 100포인트 단위로 지급 됩니다. (500MB이하 미제공)
    예시 : 남은 데이터 1.2GB ÷ 500MB X 100포인트 = 200포인트 (남은데이터 1GB로 책정 / MB단위 절사)
    통신사를 바꾸거나 해지한 경우, 보상 포인트는 제공되지 않습니다.
  • 이십세 요금제 외 일반 요금제로 변경하신 경우에는 보상 포인트가 제공되지 않습니다.
  • 요금을 미납 하신 경우에는 보상 포인트가 제공 되지 않습니다.
  • 포인트 지급일 : 사용월 + 2개월 후 지급
