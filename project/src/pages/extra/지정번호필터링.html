통신 부가 서비스

지정번호필터링
받기 싫은 전화번호를 100개까지 차단해주는 서비스
월 2,200원

고객센터 (자사휴대폰 114 / 유선 1644-5353)로 전화 문의 또는 가까운 유플러스 직영매장에서 방문 신청 가능합니다.

서비스 특징

받기싫은 전화번호를 100개까지 차단하는 서비스입니다.
통화를 원치않는 번호를 등록 후 상대방에게 송출되는 차단멘트(결번안내, 전원off, 수신불가)도 선택이 가능하며, 차단된 내역을 다음날 오전 9시에 SMS로 요약하여 받으실 수 있습니다.

  • 최대 100개까지의 번호 음성/영상 전화 및 문자(SMS,MMS)차단 가능
  • 차단 번호에 대한 차단 이력조회 가능
  • 차단 내역을 문자로 알림(다음날 오전 9시) 기능 제공
  • 차단 설정 시 상황에 맞게 멘트 설정 가능(없는번호 / 전원off / 수신불가)

이용요금

  • 월 2,200원(부가세 포함)

신청방법

  • 고객센터

    자사 휴대폰 114 (무료)
    1644-5353 (유료)

  • 유플러스 직영매장

    유플러스 직영매장에서도 방문신청 가능

    U+직영매장 찾기

U+통화도우미 APP 다운로드

구글 Play 스토어, 앱스토어 접속 > "통화도우미" 검색 > 앱 다운로드
통화 도우미 앱 접속 → 지정번호 필터링 → 지정번호 필터링 설정 → 차단 번호 ‘추가’ 선택 → 차단 번호 및 안내 멘트 입력하기 → 저장

  • 필터링 안되는 경우
    안드로이드폰의 ‘채팅플러스’와 아이폰의 ‘아이메시지’로 오는 문자메시지는 차단할 수 없습니다.
    지정번호로 오는 모든 문자메시지를 완벽하게 차단하려면 해당 기능을 꺼주세요.
  • 채팅플러스 끄는 방법
    바탕화면에 있는 메시지 앱 실행 → 오른쪽 상단 [?] → [설정] → [채팅+ 설정] → [채팅+ 관리] → [채팅+ 끄기] 누르기
  • 아이 메시지 끄는 방법
    바탕화면에 있는 설정 앱 실행 → [메시지] → ‘iMessage’ 오른쪽 버튼 눌러 끄기
  • 별도의 해지 요청이 있을 때까지 매월 자동 연장되며, 청구서를 통해 청구비용을 확인하실 수 있습니다.
  • 차단 번호는 최대 100개까지 등록 가능합니다.
  • 원하실 경우 차단전화 통보 메시지 기능이 제공됩니다.
  • 지정번호필터링 서비스와 자동응답 서비스는 동시 가입이 불가합니다.
  • 차단 멘트 설정은 없는번호, 전원off, 수신불가 (3종류)로 설정 가능합니다.
  • 서비스를 해지한 당일 재가입은 불가능합니다. (익일 재가입 가능)
