U+통화도우미 APP 다운로드

  • 구글 Play 스토어, 앱스토어 접속 > “통화도우미” 검색 > 앱 다운로드
  • 통화 도우미 앱 접속 → 앱 실행 → 음성사서함 선택 → Check Box 선택 → 음성사서함전환 선택 → 음성사서함 전환 옵션 선택

음성사서함 접속

  • 본인 휴대폰으로 이용할 경우 : *88 → 음성통화 또는 영상통화 버튼
  • 타인 휴대폰 또는 유선 전화를 이용할 경우 : 본인 번호로 전화 → 음성사서함 연결 → 숫자 3과 비밀번호 입력
  • 음성메시지 청취 후, 저장한 메시지의 보관기간은 7일이며, 최대 저장 개수는 20개 입니다. - 보관 기간 내 재 청취 후 다시 보관하면 지속적으로 7일간 연장보관 가능 - 저장된 메시지 20건 초과 시, 신규 메시지 녹음 불가
  • 삭제한 음성메시지는 복구가 불가능합니다.
  • 인사말 녹음 시간은 3분 이며, 음성메시지 녹음시간은 5분 입니다.
  • 미 청취 메시지는 3개월간 저장되며, 청취 후 저장하지 않은 메시지는 3일간 저장됩니다.
  • 최초 비밀번호는 휴대폰 번호 뒤 4자리로 되어 있으므로, 고객정보 보호를 위해 다른 번호로 변경이 필요합니다.
  • 음성메시지 청취 후, 저장되어 있는 음성메시지를 타인에게 전달해 주는 ‘메시지 전달’ 기능은 지원되지 않습니다.
