통신 부가 서비스

U+zone
LG유플러스 와이파이 공유기가 있는 곳 어디서나 와이파이를 이용할 수 있는 서비스
무료

U+유모바일 요금제 가입 완료 시 별도의 신청 없이 U+zone에 자동으로 연결 가능합니다.

서비스 특징

LG U+의 와이파이 핫스팟과 U+와이파이100 등 LG U+의 AP가 설치된 전국의 145만여개 장소에서 최대 100Mbps 속도로 스마트폰과 패드에서 무선 인터넷을 이용할 수 있는 서비스입니다.

  • LG U+ 와이파이 공유기가 있는 전국 어디서나 최대 100Mbps 속도로 연결
  • U+유모바일 고객은 별도 가입없이 무료로 이용 가능
  • 최신 무선보안기술을 적용해 무선랜 보안사고 걱정 없이 사용

  • 무료

신청방법

유모바일 요금제 가입 완료 시 별도의 신청 없이 U+zone에 자동으로 연결 가능합니다.
자동 연결 불가 시 ‘유심 이용 휴대폰 등록’을 통해 휴대폰 정보를 등록해주세요. 유심 이용 휴대폰 등록 일부 유심 사용 고객은 자동 연결이 불가할 수 있습니다.

접속방법

와이파이 연결화면의 사용 가능한 네트워크 목록 중 “U+zone” 선택

  • 노트북에서 이용하실 수 없습니다.
  • 단말자체에서 U+zone의 인증방식인 802.1x PEAP 인증을 지원하지 않는 경우에는 U+zone 을 사용하실 수 없습니다.
    예) 802.1x PEAP 미지원 피쳐폰
  • Android OS 2.1 이상의 스마트폰/패드/플레이어
  • iOS 3.0 이상의 스마트폰/패드/플레이어
  • Windows Mobile 6.5 이상의 스마트폰/패드/플레이어
