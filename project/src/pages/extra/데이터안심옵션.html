서비스 특징

요금제 기본 제공 데이터를 모두 사용했을 경우, 별도의 요금 없이 이메일, 웹서핑 등이 가능한 400kbps의 속도로 데이터를 이용할 수 있습니다.

  • 월 5,500원(부가세 포함)

    U+직영매장 찾기
  • 별도의 해지 요청이 있을 때까지 매월 자동 연장되며, 청구서를 통해 청구비용을 확인하실 수 있습니다.
  • 요금제에서 기본으로 제공되는 데이터 사용 초과 시,
    추가 요금 발생 없이 일반적인 데이터 서비스 이용이 가능한 속도로(최대 400Kbps) 이용이 가능합니다. 단, 데이터 속도가 조절되어 일부 대용량 서비스는 이용이 불편할 수 있습니다.

  • 데이터 서비스를 이용한 음성통화/영상통화는 할 수 없습니다.
