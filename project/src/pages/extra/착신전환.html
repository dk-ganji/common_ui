통신 부가 서비스

착신전환
휴대폰으로 걸려오는 전화, 메시지를 다른 전화나 휴대폰으로 연결 해주는 서비스
유료(상세정보참조)

서비스 특징

고객님의 휴대폰으로 걸려오는 전화와 메시지를 다른 전화나 휴대폰으로 연결할 수 있는 서비스로, 휴대폰을 분실하셨거나 두고 왔을 때 또는 전원 OFF 등의 상황에 유용하게 사용할 수 있는 부가서비스입니다.

  • 전화를 받지 못할 경우 지정된 번호로 바로 연결이 가능하며 유선/인터넷전화/이동전화로 연결 가능
  • 바로 연결 또는 미응답시 연결로 설정 가능

이용요금

  • 월 1,650원(부가세 포함)

착신전환과 매너콜을 함께 이용 시 착신전환서비스를 기본요금 880원에 제공
(단, 매너콜 기본제공 요금제 가입자 제외)

  • 착신전환 (통화와 문자) 월 1,650원

    바로연결
    • 음성 및 영상통화 전환
    • 단문메시지, 장문메시지 전환
    미응답시 연결
    • 음성 및 영상통화 전환

  • 착신전환 (통화만) 월 770원

    바로연결 / 미응답시 연결
    • 음성 및 영상통화 전환

    서비스 설정

    착신전환 기본 설정은 ‘바로연결’(즉시 착신전환) 상태입니다.

    • 바로연결 : Check Box선택 > 서비스 설정 활성화 상태 확인 > 착신전환 될 번호 입력>저장
    • 미 응답 시 연결 : Check Box 선택 > 서비스 설정 활성화 상태 확인 > 미 응답 시 연결 선택 > 설정 완료 팝업
  • 통화도우미 앱을 이용하면 24시간 내내 어디서나 서비스 설정, 변경, 조회할 수 있습니다.
  • 아이폰 이용 고객은 통화도우미 앱을 통한 가입이 불가합니다. 자세한 사항은 휴대폰에서 114(무료), 1644-5353(유료)로 문의하세요.
  • 원래 번호 외에 추가로 받은 듀얼넘버 서비스의 가상 번호는 가입할 수 없습니다.
  • 착신전환 대상 번호 설정은 하루에 20회로 제한됩니다.
  • 착신전환 무료통화 시간은 월 270분까지이며, 초과 시 부가세포함 1초당 1.98원의 추가요금이 발생합니다.
  • 선불 휴대폰(PPS)요금제 가입자는 2017년 10월 1일부터 착신전환 서비스를 이용할 수 없습니다.
