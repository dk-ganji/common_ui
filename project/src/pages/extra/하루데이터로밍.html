통신 부가 서비스

하루데이터로밍

일 11,000원

서비스 특징

171개국에서 현지수도시간 하루 기준으로 데이터 해외로밍 서비스를 이용할 수 있는 요금제입니다.

  • 해외에서 LTE 및 3G 데이터를 저렴하게 이용
  • 여행기간에 맞춰 주요 171개국에서 이용
  • 방문국가의 현지 수도시간을 기준으로 밤 12시까지 하루로 계산
  • 일 300MB 다 쓰면 200Kbps 속도로 무한 사용
  • 가입 후 해지하지 않으면 서비스를 이용할 수 있는 국가에서 자동 적용

이용요금

  • 11,000원/1일(부가세 포함)
  • 해외로밍 데이터를 사용하는 시점부터 현지 수도시간 1일 기준으로 요금이 발생됩니다.
  • 요금제 가입 후 데이터 서비스를 사용하지 않는 경우 요금이 청구되지 않습니다.
  • 데이터 해외로밍 전용 서비스로 음성통화, 문자메시지를 이용하는 경우 추가요금이 발생합니다.
  • 하루에 2개 이상의 국가에서 이용하는 경우 마지막으로 사용한 국가의 시간을 기준으로 합니다.
  • 해외로밍 이용 요금은 국내 통신료와 별도의 항목으로 구분되어 1개월 이후 청구서에 반영됩니다.

할인사업자 수동 설정 안내

해외로밍 이용 시 LG유플러스와 계약한 현지 사업자 통신망으로 자동 연결됩니다.

네트워크 상황에 따라 연결이 원활하지 않은 경우 아래와 같은 방법으로 사업자를 직접 선택할 수 있습니다.

  • LG폰, 삼성폰 등 안드로이드폰
    휴대폰 → 설정 → 해외로밍 → 해외 이동통신사 설정 → ‘이동통신사 자동 설정’ 끄기를 누른 후 하단에 보이는 해외사업자를 직접 선택하여 이용할 수 있습니다.
  • 애플 아이폰
    휴대폰 → 설정 → 네트워크 사업자 → ‘네트워크 사업자’의 ‘자동’ 끄기를 누른 후 하단에 보이는 해외사업자를 직접 선택하여 이용할 수 있습니다.
