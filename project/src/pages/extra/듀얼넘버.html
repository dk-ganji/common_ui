통신 부가 서비스

듀얼넘버
하나의 휴대폰으로 2개의 번호를 이용할 수 있는 서비스
월 3,300원

고객센터 (자사휴대폰 114 / 유선 1644-5353)로 전화 문의 또는 가까운 유플러스 직영매장에서 방문 신청 가능합니다.

서비스 특징

하나의 휴대폰을 원번호 이외에 별도의 번호를 추가하여, 하나의 휴대폰을 두 개의 폰처럼 사용할 수 있는 서비스입니다.

  • 하나의 휴대폰에 2개의 번호로 통화, 문자 외 SNS, 카카오톡 등을 이용할 수 있는 서비스
  • 교사, 세일즈맨, 배달업 종사자 등 많은 사람과 전화번호 를 공유해야 하는 고객님께 추천
  • 휴대폰 멀티 계정 기능을 통해 메신저 앱(카카오톡 등) 2개 설치하여 이용 가능

이용요금

  • 월 3,300원(부가세 포함)

신청방법

  • 고객센터

    추가 문의는 고객센터에서 전화상담 가능
    자사 휴대폰 114 (무료)
    1644-5353 (유료)

  • 유플러스 직영매장

    유플러스 직영매장에서도 방문신청 가능

    U+직영매장 찾기

U+통화도우미 APP 다운로드

구글 Play 스토어 접속 > “통화도우미” 검색 > 앱 다운로드 (아이폰의 통화도우미에서는 듀얼넘버 메뉴를 지원하지 않습니다.)

  • 듀얼넘버 자동발신 기능은 2020년 12월 1일부터 제공되지 않습니다.
    자동 발신 기능 : 상대방 번호 뒤에 #을 추가하지 않아도 모든 전화가 듀얼넘버로 발신되는 기능

듀얼넘버로 전화하기

1. U+통화도우미 앱 내 주소록을 활용하는 방법

1단계

U+통화도우미 앱 실행 후 [듀얼넘버] 누르기

2단계

‘원번호 친구 목록’에서 상대방 이름 오른쪽 + 버튼 누르기

3단계

팝업창에서 [확인] 버튼 누르기

4단계

상대방이 ‘듀얼넘버 친구’ 목록에 추가되면 이름 옆버튼 누르기

5단계

원래 번호가 아닌 듀얼넘버로 전화 발신

2. 상대방 번호를 직접 입력하는 방법

  • 안드로이드 휴대폰 : 상대방 전화번호 + # + 통화 (2G 휴대폰은 *77# + 상대방 전화번호 + 통화)
  • 아이폰 등 외산 휴대폰 : 상대방 전화번호 + * + 통화

듀얼넘버로 문자메세지 보내기

1. U+통화도우미 앱 내 주소록을 활용하는 방법

일부 구형 단말에서는 통화도우미 앱의 듀얼넘버 문자메시지 발신기능을 제공하지 않을 수 있습니다.

1단계

U+통화도우미 앱 실행 후 [듀얼넘버] 누르기 > ‘원번호 친구’ 목록에서 상대방 이름 옆 + 버튼 누르기

2단계

상대방이 ‘듀얼넘버 친구’ 목록에 추가되면 이름 옆버튼 누르기

3단계

메시지 입력 후 듀얼넘버로 문자메시지 보내기

2. 상대방 번호를 직접 입력하는 방법 1

안드로이드 단말에서만 이용 가능한 방법입니다.

1단계

휴대폰 바탕화면에 있는 전화 앱 누르기

2단계

상대방 전화번호 입력 후 마지막에 #버튼 누르기 > 우측 상단의버튼 누르기

3단계

우측 상단의버튼을 눌러서 나온 메뉴 중 [메시지 보내기] 버튼 누르기

4단계

메시지 입력 후 듀얼넘버로 문자 발신

3. 상대방 번호를 직접 입력하는 방법 2

  • 안드로이드 휴대폰 : 상대방 전화번호 + # + 통화 (2G 휴대폰은 *77# + 상대방 전화번호 + 통화)
  • 아이폰 등 외산 휴대폰 (#으로 발신이 불가능한 휴대폰) : 상대방 전화번호 뒤에 *버튼을 누르기

듀얼넘버 바꾸기

1단계

듀얼넘버 변경하는 화면으로 이동하기
U+통화도우미 앱 > 듀얼넘버 > 메뉴 > 듀얼넘버 변경

2단계

바꾸고 싶은 휴대폰 번호 뒷자리를 입력한 후 ‘번호조회’ 누름

3단계

변경할 수 있는 번호 목록에서 원하는 번호 선택 후 ‘설정’ 누름

4단계

번호변경 확인 후 완료

듀얼넘버로 수신되는 전화와 문자메시지 구분하기

전화받기

걸려온 전화번호 뒤에 ‘#’이 표시됩니다. (아이폰에서는 걸려온 전화번호 뒤에 ‘*’이 표시 됩니다.)

문자메시지 받기

보낸 사람의 전화번호 뒤에 ‘#’이 표시되고, 문자메시지 본문의 내용 앞에 ‘듀>’가 표시됩니다.

  • 안드로이드 휴대폰 이용 고객은 통화도우미 앱을 통해 간편하게 가입할 수 있습니다.
  • 듀얼넘버 가입 및 듀얼넘버 변경 시 두 개의 번호를 적용하기 위해서는 휴대폰의 전원을 껐다가 다시 켜야 합니다.
  • 아이폰 이용 고객은 통화도우미 앱을 통한 가입이 불가합니다. 자세한 사항은 휴대폰에서 114(무료), 1644-5353(유료)로 문의하세요.
  • 듀얼넘버 자동발신 기능은 2020년 12월 1일부터 제공되지 않습니다.
  • 별도의 해지 요청이 있을 때까지 매월 자동 연장되며, 청구서를 통해 청구비용을 확인하실 수 있습니다.
  • 듀얼번호로는 부가서비스 가입이 불가 하며, 원번호에 가입되어 있는 부가서비스가 있을 경우 듀얼번호로도 동일하게 적용 됩니다.
  • 듀얼번호 변경 90일 이내 최대 2회 가능합니다. (미래부 번호관리 정책에 따라 2017년 1월부로 변경됨)
  • 서비스 해지 후 재가입은 익월에 가능합니다.
  • 원넘버 서비스와 동시 가입이 불가합니다.
  • 음성사서함으로 수신된 음성 메시지에 대해 원번호/듀얼번호 구분 되지 않습니다.
  • 해외 수신 및 발신 시 듀얼넘버 서비스 이용이 제한될 수 있으며, 상대방에게 원번호가 노출될 수 있습니다.
  • #0000 (방송, 행사 등) 형태의 문자메시지 서비스가 제공되지 않으며, 문자메시지 전송 시 원 번호가 노출될 수 있습니다.
  • 듀얼번호에 대하여 음성사서함으로 전환하는 기능은 LTE 이용자일 경우 제공되지 않습니다.
  • 톡톡 듀얼넘버를 선호번호로 이용하실 경우 선호번호 정책에 따라 1인당 1개의 번호만 부여되며, 취득 후 1년간 재 취득제한 및 선호번호 이용 중 추가취득이 불가합니다.
  • 듀얼번호로의 문자 채팅 기능은 지원하지 않습니다.
