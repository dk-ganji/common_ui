서비스 특징

83개국에서 사용시작시간을 설정하여 해외로밍 데이터와 주고 받는 음성 전화를 무제한으로 이용할 수 있는 요금제입니다.

  • 받는 음성 전화와 거는 음성전화 무제한, 해외로밍 데이터 무제한(속도제한)으로 이용 가능
  • 다수 사용자의 정상적인 사용을 보호하기 위해 하루 누적 사용량이 4GB를 초과한 경우 이용속도가 400kbps로 제공
  • 테더링(데이터 함께 쓰기)용 데이터 하루 4G 별도 제공. 다 쓰면 400kbps 속도로 무제한
  • 한국, 방문국가, 다른 국가 모두 음성 전화를 무제한으로 이용
  • 고객이 설정한 사용시작 시각부터 24시간 단위로 이용

할인사업자 수동 설정 안내

해외로밍 이용 시 LG유플러스와 계약한 현지 사업자 통신망으로 자동 연결됩니다.

네트워크 상황에 따라 연결이 원활하지 않은 경우 아래와 같은 방법으로 사업자를 직접 선택할 수 있습니다.

  • LG폰, 삼성폰 등 안드로이드폰
    휴대폰 → 설정 → 해외로밍 → 해외 이동통신사 설정 → ‘이동통신사 자동 설정’ 끄기를 누른 후 하단에 보이는 해외사업자를 직접 선택하여 이용할 수 있습니다.
  • 애플 아이폰
    휴대폰 → 설정 → 네트워크 사업자 → ‘네트워크 사업자’의 ‘자동’ 끄기를 누른 후 하단에 보이는 해외사업자를 직접 선택하여 이용할 수 있습니다.
