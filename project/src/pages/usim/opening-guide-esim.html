eSIM 가입 안내

휴대폰 하나로 2개의 번호를 사용하세요!

유심 카드를 구입하지 않아도 쉽고 빠르게 셀프개통을 할 수 있어요.

eSIM 이란?

휴대폰 단말에 내장되어 있는 SIM으로 디지털 SIM을 말합니다.
eSIM은 기존의 물리적인 USIM칩을 끼우는 작업 없이 사업자가 제공하는 QR코드를 활용한 eSIM 활성화 단계를 거쳐 휴대폰을 사용하실 수 있습니다.

유심칩 구매 불필요

eSIM은 스마트폰에 내장된 SIM으로 개통하기 때문에 별도 유심
구매를 하지 않아도 됩니다.

eSIM 다운로드 가격 : 2,750원 최초 1회 무료 (익월 요금명세서에
포함되어 청구됩니다.)

배송/픽업 불필요

eSIM은 스마트폰에 내장되어 있기 때문에 배송/바로픽업이
불필요 합니다.

휴대폰에 유심 장착없이 바로 사용 가능

eSIM 가입 완료 시 요금 정보 프로파일을 다운받으면 바로 사용이
가능합니다.

프로파일 다운로드 실패 시, 고객센터로 문의주세요.
(1644-5353)

eSIM 개통 방법

아래 3가지 방식 중 1가지로 개통해주세요. 휴대폰 정보가 불일치 할 경우 개통이 되지 않습니다. 한개의 휴대폰에 동의 명의로만 개통이 가능해요. (본인+타인/개인+법인/개인+개인사업자 조합 불가)

아이폰

  • 1. eSIM 개통할 아이폰에서 오는
    Push 메세지를 눌러주세요.

  • 2. 셀룰러 요금제를 설치해주세요.
    (셀룰러 요금제 설치 후 계속 버튼 탭)

  • 3. 새로 개통할 eSIM 요금제
    레이블을 선택해주세요.

  • 4. 기본회선을 선택하고
    계속을 눌러주세요!

  • 5. 애플 ID를 연동할 회선을
    선택하고 계속을 눌러주세요.

  • 6. 셀룰러 데이터에 사용할 회선을
    선택 후 계속을 누르면 끝!

안드로이드

  • 1.휴대폰 [설정] 메뉴의 [연결]을
    눌러주세요.

  • 2. [SIM 카드 관리자] 메뉴를
    눌러주세요.

  • 3. [모바일 요금제 추가]
    메뉴를 눌러주세요.

  • 4. 요금제가 추가 되는 동안
    잠시만 기다려주세요.

  • 5. 애플 ID를 연동할 회선을
    선택하고 계속을 눌러주세요.

eSIM 개통할 휴대폰으로 QR코드를 스캔해주세요.

아이폰 QR코드 스캔은 어떻게 하나요?

  • 1. 카메라로 QR코드를
    스캔해주세요.

  • 2. [계속] 버튼을 클릭해주세요.

  • 3.[셀룰러 요금제 추가]
    버튼을 클릭해주세요.

안드로이드 QR코드 스캔은 어떻게 하나요?

  • 1.카메라로 QR코드를 스캔하고
    [옵션보기]를 클릭해주세요.

  • 2.[휴대전화에 요금제 추가]를
    눌러주세요.

  • 3.[확인] 버튼을 눌러주세요.

고객님의 활성화 코드를 eSIM을 개통할 휴대폰에 입력해주세요.

lgu.prod.ondemandconnectivity.com

LPA:1$lgu.prod.ondemandconnectivity.com$
아이폰 활성화 코드 입력 방법

  • 1. 아이폰 [설정]메뉴의 [셀룰러]
    메뉴를 눌러주세요.

  • 2.[셀룰러요금제추가]메뉴를 누른 후 [세부사항 직접 입력]
    을 눌러주세요.

  • 3.SM+DP+주소를 입력해주세요.

안드로이드 활성화 코드 입력 방법

  • 1.휴대폰 [설정]메뉴의
    [연결]을 눌러주세요.

  • 2.[SIM 카드 관리자] 메뉴를
    눌러주세요.

  • 3.[모바일 요금제 추가] 메뉴를
    눌러주세요.

  • 3.[통신사 QR코드 스캔]을
    눌러주세요.

  • 3.[개통코드입력]을 눌러주세요.

  • 3.활성화 코드를 입력해주세요.
