휴대폰 고유번호 확인 방법
  • *#06# 입력
  • 휴대폰 설정 정보

통화 키패드에서 *#06# 을 입력하면 휴대폰 고유번호를 확인할 수 있습니다.

화면 캡쳐 후 업로드 시 휴대폰 고유번호가 자동 입력됩니다.

EID 정보가 없다면 ‘유심 가입하기’를 이용해 주세요.

화면 캡쳐 방법

아이폰
  • FACE ID 기능 있는 경우 : 전원 버튼과 음량 높이기 버튼을 동시에 누름
  • TOUCH ID 기능 있는 경우 : 전원 버튼과 홈 버튼을 동시에 누름
갤럭시
  • 측면 버튼과 음량 줄이기 버튼을 동시에 누름
  • 화면을 왼쪽에서 오른쪽으로 쓸어 내림

휴대폰 설정 정보에서 고유번호를 확인할 수 있습니다.
EID, IMEI1, IMEI2를 찾아주세요.

아이폰

설정 > 일반 > 정보

갤럭시

설정 > 휴대전화 정보 > 정보
