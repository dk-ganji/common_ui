  • 분실정지는 월 2회까지 가능하며, 초과하는 경우 고객센터 114(무료) 또는 1644-5353(유료)으로 문의하세요.

  • 분실신고를 악용하여 무료 통화 등을 하는 경우 정상 요금이 청구되며 법적 책임을 물을 수 있습니다.
  • 분실등록 후 일시 정지를 하지 않으면 휴대폰 기본료가 그대로 청구됩니다.
  • ‘분실폰 위치찾기’는 분실폰 등록 후에 조회가 가능합니다.
  • 분실폰에 문자보내기 선택 시, 분실 된 휴대폰 습득자로부터 연락을 받을 수 있도록 문자 메시지가 발송됩니다.
  • 일시정지를 설정하는 경우 부가세 포함 금액 월 4,400원의 기본 요금이 청구됩니다.
  • 월 중간에 일시정지를 설정하는 경우 설정 이전까지는 이용중인 요금제의 기본 요금을 사용 일수 만큼 청구하고, 정지 기간 중에는 부가세 포함 금액 월 4,400원의 기본 요금을 사용 일수 만큼 청구 합니다.

  • 휴대폰 분실신고 시 타 사용자의 USIM 사용은 불가합니다.
